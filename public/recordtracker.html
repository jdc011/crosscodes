Record Tracker


Synopsis
This is a command line application designed for UNIX platforms. A user will be able to store and retrieve recorded information. Information is saved to the hard disk. A user also has the option to clear all saved information. This app is a very basic data storage implementation that can be extended to other applications.

Details
Written in C++ along with file I/O capability, a user can create, store, and delete information on the hard disk. This file I/O implementation is used for the information saving feature so a a user is able to retrieve written information after the application has been turned off. Information is written to a file that is read by the application with a real-time updating occupancy status. A developer can extend this basic implementation of data storage and retrieval via linked list to more complex database style applications.
